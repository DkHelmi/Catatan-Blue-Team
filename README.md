# Catatan Blue Team

Kumpulan artikel dan catatan seputar keamanan defensif (blue team, SOC, DFIR, threat hunting) yang ditulis dari sudut pandang praktisi dalam Bahasa Indonesia. Tujuannya berbagi cara berpikir dan pola kerja, bukan sekadar ringkasan teori, dengan rujukan ke sumber resmi seperti NIST, SANS, MITRE, dan Microsoft.
